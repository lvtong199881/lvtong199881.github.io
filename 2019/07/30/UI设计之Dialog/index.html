Dialog概述

dialog是应用向用户展示信息并提供一些的操作的一种通知形式，与toast不同，dialog会获取界面焦点，中断用户操作，这种方式比较突兀，用户体验不是很好，除了像登录场景、检测到用户作弊等情况，其他情况下还是要谨慎使用，更加推荐使用snackbar。
